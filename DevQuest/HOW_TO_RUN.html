⚡ HOW TO RUN DEVQUEST CORRECTLY

Read this if you're seeing "Cannot GET /detail.html"

❌ THE PROBLEM

You downloaded TWO folders:
devquest/ ← OLD version (v1) — missing detail.html
devquest2/ ← NEW version (v2) — has ALL files ✅

You are opening devquest instead of devquest2.
✅ THE FIX — Follow these exact steps:
Step 1 — Close VS Code completely
Press Alt+F4 or File → Exit. Make sure it's fully closed.
Step 2 — Open VS Code fresh
Open VS Code from your desktop or Start menu.
Step 3 — Open the CORRECT folder
In VS Code: File → Open Folder

Navigate to where you downloaded the files and select:
devquest2

⚠️ NOT devquest — that's the old one!
Step 4 — Check the Explorer panel
In VS Code's left sidebar (Explorer), you should see EXACTLY these 7 files:

app.js
checklist.html
detail.html ← if you don't see this, you're in the wrong folder!
index.html
projects.html
results.html
style.css
Step 5 — Right-click index.html → Open with Live Server
In the Explorer panel, right-click on index.html
Click "Open with Live Server"

Your browser should open: http://127.0.0.1:5500/index.html
✅ That's it! The app should now work fully.
